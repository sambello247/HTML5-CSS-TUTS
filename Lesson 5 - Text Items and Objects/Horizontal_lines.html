In the early days of web design, pages were built to target a particular screen size. If the user had a larger or smaller screen than the designer expected, results ranged from unwanted scrollbars to overly long line lengths, and poor use of space.

As more diverse screen sizes became available, the concept of responsive web design (RWD) appeared, a set of practices that allows web pages to alter their layout and appearance to suit different screen widths and resolutions.

In other words, Responsive Web Design isn’t a separate technology — it is an approach or a term used to describe a set of best practices used to create a layout that can respond to any device.
